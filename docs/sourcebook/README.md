# Sourcebook

Standards of record and canonical definitions for SPECTRO. These pages fix the meaning of terms and equations used by the timeline, solver, dispatcher, and event manager.

**Required standards:** The improvising standards for the app are **required** and are **recorded for their entireties**. See [required_standards.json](required_standards.json) for the full manifest and [improvising_standards.md](improvising_standards.md) for the umbrella list.
